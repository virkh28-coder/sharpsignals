## You handle vs Claude handles

**You (human-required):**
- Pick brand name → register domain + IG + Telegram handles
- Create IG Business account + Meta Developer App (for posting API)
- Apply to sportsbook affiliate programs
- Set up Substack / Ghost account
- Add API keys to `.env`
- Review + approve first ~10 posts before automation runs unattended

**Claude Code (everything else):**
- All scraping + data pipelines
- Model training + prediction
- Pick generation + logging
- Caption + graphic generation
- IG / Telegram posting
- Track record publishing
- Newsletter drafting
- Compliance checks
