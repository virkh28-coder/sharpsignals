# Brand Name Shortlist

## How to verify availability

For each candidate, check before buying:

1. **Domain** → https://porkbun.com or https://namecheap.com
2. **Instagram** → https://www.instagram.com/{HANDLE}/ (404 page = available)
3. **Telegram** → https://t.me/{HANDLE} (invalid link = available)
4. **TikTok** → https://www.tiktok.com/@{HANDLE}
5. **Trademark (US)** → https://tmsearch.uspto.gov
6. **Trademark (CA)** → https://www.ic.gc.ca/cipo/

**Pick the first candidate where .com + IG + Telegram are all open.**

## Selection criteria

- 4–12 characters, pronounceable
- Evokes algorithm / data / edge / analytics
- Not a person's name (faceless brand principle)
- Neutral across all 5 sports
- Works internationally (cricket → huge India market)

## Never pick

- ❌ VIP / Premium / Lock / Guarantee / Insider / Secret / Fixed
- ❌ A person's name (defeats faceless-brand purpose)
- ❌ "Tips" / "Tipster" (scamdicapper-coded)
- ❌ Gimmicks: BetGod, WinKing, ProfitMaster, etc.
