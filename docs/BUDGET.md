# Budget — First Year

## Target: under $2,000 total

## One-time setup (Month 0)

| Item                           | Cost    |
| ------------------------------ | ------- |
| Domain (sharpsignals.com/.bet) | $12–40  |
| Logo design (Fiverr or AI)     | $0–50   |
| Legal review of ToS/Privacy    | $0–150  |
| **Subtotal**                   | **$12–240** |

Logo can be generated for $0 using Claude + SVG, or $20–50 on Fiverr if you want human polish.
Legal review optional at launch; strongly recommended before taking paid subs.

## Monthly recurring

| Item                          | $/mo         |
| ----------------------------- | ------------ |
| The Odds API (20K reqs)       | $30          |
| Sportmonks Cricket            | $19–29       |
| Anthropic API (content gen)   | $50–100      |
| VPS (Hetzner CX11 or Fly.io)  | $5–10        |
| Substack                      | $0 (10% cut) |
| Ghost Pro (if not Substack)   | $11          |
| **Subtotal**                  | **$104–180**     |

Annual run-rate: **$1,248–2,160**.

Choose Substack (no monthly fee, 10% revenue share) for first 6 months — switch to Ghost later once you're over ~$500 MRR.

## First-year total

| Scenario      | Total     |
| ------------- | --------- |
| Lean setup    | **$1,260** |
| Standard      | **$1,590** |
| With buffer   | **$1,900** |

Leaves ~$100–700 runway for:
- First paid ad test ($50–100 on Meta, AGCO-compliant geo-targeting Ontario only)
- Additional data sources if a sport underperforms
- Domain renewal + legal review

## What we deliberately do NOT pay for in year 1

- ❌ Paid IG followers / engagement pods (violates TOS, useless)
- ❌ "Growth hacking" services
- ❌ Paid sports data beyond The Odds + Sportmonks (free sources are sufficient for v0)
- ❌ Office, team, contractors (you + Claude = the team)
- ❌ Stock video footage (we use our own graphics, charts, screen recordings)
